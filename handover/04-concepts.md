# 04-concepts.md

ドメイン用語の定義。`Not`欄には、混同してはいけない概念を明記する。

## Format

```
Concept: [用語]
Definition: [定義]
Aliases: [別名があれば]
Not: [混同してはいけない概念]
Related Concepts: [関連する用語]
```

## Entries

### Memory（記入例）
- **Definition**: アプリ内の分析モジュールが、ユーザーの人生の重要
  イベントを抽出・保持する機能（例：Memory Module）
- **Aliases**: なし
- **Not**: 02-philosophy.mdの「Developer Preferences / Working Memory」
  （開発者自身の作業スタイルの記憶）と混同しないこと。両者は
  全く別の対象を指す
- **Related Concepts**: Pattern Module, life_events
